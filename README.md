# 人脸识别系统简介 python865

## 项目概述

本项目是基于Python、OpenCV和dlib技术实现的人脸识别系统，旨在提供高效、实时的人脸检测与识别功能。系统采用Pyqt5框架进行界面设计，确保用户操作的便捷性。

## 技术栈

- **编程语言：** Python
- **人脸识别算法和框架：** dlib、OpenCV
- **界面设计框架：** Pyqt5

## 功能模块

### 人脸录入

- 通过摄像头实时录入人脸信息。
- 默认录入人脸20张，可在代码中进行修改。

### 人脸模型训练

- 对录入的人脸进行模型训练，以便进行准确识别。

### 人脸实时识别

- 实时检测摄像头捕捉到的人脸，并进行身份识别。

## 系统角色

- **用户：** 通过界面操作进行人脸录入和识别。
- **系统：** 负责处理图像数据，执行人脸识别算法。

## 运行环境

- 操作系统：兼容Windows、Linux等常见操作系统
- Python版本：3.x及以上
- 必要依赖库：opencv、dlib、Pyqt5
